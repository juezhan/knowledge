## 提交新代码
1. cd 本地项目文件夹
2. git init
3. git add .
4. git commit -m "提交注释"
5. git remote add origin https://github.com/juezhan/项目名.git
6. git push -u origin master	// 提交到主分支
7. git config user.email "注册邮箱"
8. git config user.name "用户名"
